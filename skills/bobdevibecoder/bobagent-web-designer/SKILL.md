# Web Designer Skill

Professional website design service powered by AI. Generates production-ready Next.js + Tailwind CSS code.

## Commands

### `generate <template> <brief>`
Generate a complete website from a template and client brief.

**Templates:**
- `landing` - SaaS/product landing page (dark glassmorphism theme)
- `portfolio` - Personal/agency portfolio site
- `blog` - Blog/content site with article layout
- `docs` - Documentation site
- `pricing` - Pricing page with tier cards
- `saas` - Full SaaS marketing site (hero + features + CTA + footer)

**Example:**
```
web-designer generate saas "A crypto portfolio tracker called CoinView with purple/blue theme. Features: real-time prices, portfolio tracking, alerts, API access."
```

### `templates`
List all available templates with descriptions and pricing.

### `preview <template> <brief>`
Generate a text description of what the design would look like without generating code.

### `quote <brief>`
Generate a professional price quote for a client request.
