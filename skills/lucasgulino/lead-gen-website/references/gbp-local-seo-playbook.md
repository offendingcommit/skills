# GBP Playbook (Local SEO) — Relevance / Distance / Prominence

Objectif : gagner le **Local Pack** et la confiance.

## 1) Relevance (pertinence)
- Catégories : 1 primaire + 2–4 secondaires (cohérentes)
- Services : liste détaillée (libellés naturels)
- Description : claire + zone + transparence (si mise en relation)
- Q/R : répondre aux questions réelles (prix, délais, marques)

## 2) Distance (proximité)
- Zone : définir la zone de service proprement
- Cohérence : site + GBP + citations NAP alignés

## 3) Prominence (notoriété)
- Avis :
  - demander après service (SMS/WhatsApp)
  - viser avis détaillés (type de panne + commune + satisfaction)
  - répondre à tous les avis
- Photos :
  - 5–10 photos réelles (avant/après, véhicule, matériel, zone)
  - 1 post GBP / semaine au départ
- Citations NAP : 5–15 sources locales/sectorielles de qualité (pas de spam)
- Prescripteurs : syndics, agences immo, conciergeries, menuisiers/électriciens

## Plan 30 jours (simple)
S1 : GBP complet + 10 photos + 5 Q/R + 2 posts
S2 : 3 citations NAP + 5 avis demandés + 2 posts
S3 : 2 partenariats prescripteurs + 2 posts + album avant/après
S4 : audit catégories/services + 3 nouvelles Q/R + refresh description

## Garde-fou leadgen
Si mise en relation :
- Clarifier le statut dans la description et sur le site
- Éviter toute présentation trompeuse (adresse, équipe, interventions)
