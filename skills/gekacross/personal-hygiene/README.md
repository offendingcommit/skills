# Personal Hygiene

Помощник по уходу за собой для OpenClaw. Трекает рутины, продукты, стрижки, визиты к стоматологу.

## Что умеет

- Хранить утреннюю и вечернюю рутины
- Трекать продукты и средства ухода
- Напоминать когда пора к барберу
- Отслеживать визиты к стоматологу
- Вести список покупок по уходу

## Команды

```
моя рутина                     — показать утреннюю/вечернюю рутину
добавь в рутину [действие]     — обновить рутину
стрижка сегодня                — записать дату стрижки
когда стричься                 — напомнить когда пора
стоматолог сегодня             — записать визит
купить [продукт]               — добавить в список
мои продукты                   — показать список средств
инструкция                     — показать все команды
```

## Установка

```bash
clawdhub install personal-hygiene
```

## Данные

Хранятся в `knowledge/personal/hygiene.md` в воркспейсе агента.
