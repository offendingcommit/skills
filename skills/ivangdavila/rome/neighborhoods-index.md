## Area Groups

### Historic Center
- **Centro Storico** — Tourist heart, Pantheon, Piazza Navona
- **Trastevere** — Charming streets, nightlife
- **Monti** — Trendy, Colosseum-adjacent
- **Campo de' Fiori / Jewish Ghetto** — Market, history
→ See `neighborhoods-historic.md`

### Upscale Residential
- **Parioli** — Wealthy, diplomatic, green
- **Prati** — Vatican-adjacent, quiet professional
- **Aventino** — Peaceful hilltop, views
- **Trieste-Salario** — Upper-middle class
→ See `neighborhoods-upscale.md`

### Working/Middle Class
- **Testaccio** — Foodie capital, authentic
- **San Giovanni** — Local, practical
- **Monteverde** — Green, family
- **San Lorenzo** — Student, affordable
→ See `neighborhoods-residential.md`

### Outer & Emerging
- **EUR** — Fascist architecture, modern
- **Garbatella** — Working class charm
- **Pigneto** — Hipster emerging
- **Ostiense** — Industrial, nightlife
→ See `neighborhoods-outer.md`
