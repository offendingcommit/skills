# Visitor Tips — Barcelona

## Transport Basics

### Payment
- **T-Casual** (10 trips) = best for most visitors
- **Hola BCN!** (1-5 days unlimited) = if using transport heavily
- Contactless works but more expensive per trip
- Airport metro (L9 Sud) has supplement

### Best Apps
- **TMB App** — Official metro/bus, real-time
- **Citymapper** — Best journey planning
- **Moovit** — Alternative planner
- **Bicing** — E-bike sharing (residents only)

### Tips
- Barcelona is very walkable — center fits in 45-minute diameter
- Avoid metro at rush hour (8-9:30am, 6-8pm)
- Airport train (Rodalies R2) cheaper than metro
- Taxi/Uber from airport ~€40-45 to center

## Pickpocket Prevention (CRITICAL)

Barcelona has Europe's highest pickpocketing rate for tourists. Take this seriously.

### High-Risk Areas
- Las Ramblas
- Metro (especially L3 between Catalunya-Diagonal)
- La Boqueria
- Gothic Quarter
- Barceloneta beach
- Any tourist attraction queue
- Sagrada Família area

### Prevention
| Tip | Why |
|-----|-----|
| **Cross-body bag, zip facing you** | Hardest to access |
| **Phone in front pocket** | Back pocket = easy target |
| **Don't stop for petition signers** | Distraction while accomplice steals |
| **Bag on lap at restaurants** | Floor or back of chair = target |
| **Watch for "helpful" strangers** | Often working with thieves |
| **No phone on restaurant table** | Grab-and-run common |
| **Backpack to front in crowds** | Or skip backpack entirely |

### Common Scams
| Scam | Response |
|------|----------|
| Deaf charity petition | "No, gràcies" and keep walking |
| Rose/bracelet sellers | Firm "No" and don't accept item |
| "Found" ring | Ignore completely |
| Bird droppings/sauce splash | Someone "helping" is robbing you |
| Metro card problems | Don't accept "help" |

**Emergency:** 112 | Non-emergency police: 092 (local), 091 (national)

## Shopping Guide

### What to Buy
- **Olive oil** (quality Catalan brands)
- **Vermouth** (local tradition)
- **Cava** (Catalan sparkling wine)
- **Espadrilles** (traditional shoes)
- **Ceramics** (not tourist shop junk)
- **Jamón** (vacuum-packed)
- **Paprika/saffron** (at Boqueria)

### Where to Shop
| Type | Where |
|------|-------|
| **Souvenirs (quality)** | El Born shops, NOT Ramblas |
| **Fashion** | Passeig de Gràcia (luxury), Gràcia (indie) |
| **Vintage** | Raval, El Born |
| **Food gifts** | Boqueria, Colmado Quílez |
| **Books** | La Central (beautiful) |
| **Design** | Vinçon, Born area |

### Avoid
- Tourist shops on Las Ramblas (overpriced junk)
- "Authentic" items at €1 shops
- Street vendors selling knockoffs

## Barcelona Quirks

| Quirk | Reality |
|-------|---------|
| **Catalan identity** | Strong regional pride. Be respectful. |
| **Siesta** | Less common than other Spain, but some shops close 2-5pm |
| **Sunday closures** | Many shops closed (though tourist areas open) |
| **Late everything** | Dinner at 10pm normal |
| **Beach culture** | Topless sunbathing normal on beaches |
| **Noise tolerance** | Barcelona is loud at night |
| **Asking for check** | You must ask. They won't bring it |
| **Tap water** | Safe, but restaurants ask "agua con/sin gas?" |
| **Strikes** | Transport strikes happen. Check news. |

## Emergency Info

| Need | Contact |
|------|---------|
| **Any emergency** | 112 |
| **Mossos d'Esquadra (Catalan police)** | 112 |
| **Local police** | 092 |
| **National police** | 091 |
| **Ambulance** | 061 |
| **Lost/stolen cards** | Cancel via app immediately |
| **Lost passport** | Police report → Consulate |
| **Hospital** | Go to Urgències (ER), free for emergencies |

### Pharmacies (Farmàcies)
- Green cross = pharmacy
- At least one open 24h per district
- Pharmacists can recommend for minor issues
- Many meds available without prescription

### Police Report for Theft
- Go to Comissaria (police station)
- Or tourist-specific: Guardia Urbana on Las Ramblas
- Get "denúncia" for insurance claims

| Do | Don't |
|----|-------|
