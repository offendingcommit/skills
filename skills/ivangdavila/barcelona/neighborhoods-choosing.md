# Choosing a Neighborhood in Barcelona

## Decision Framework

### Step 1: Define Your Budget

| Monthly Rent Budget | Best Options |
|---------------------|--------------|
| **Under €900** | Nou Barris, L'Hospitalet outskirts |
| **€900-1,200** | Sant Andreu, Sants, El Clot, L'Hospitalet center |
| **€1,200-1,500** | Poble Sec, Poblenou, Horta, Sant Antoni |
| **€1,500-1,800** | Gràcia, El Born, Eixample Esquerra |
| **€1,800+** | Eixample Dreta, Sarrià, Diagonal Mar |

### Step 2: Define Your Lifestyle

| If You Want... | Consider |
|----------------|----------|
| **Beach lifestyle** | Barceloneta, Poblenou, Vila Olímpica |
| **Village feel** | Gràcia, Sarrià |
| **Nightlife + social** | El Born, Gràcia, Raval |
| **Tech job proximity** | Poblenou (22@), Eixample |
| **Family environment** | Sarrià, Les Corts, Horta |
| **Budget + central** | Raval (upper), Poble Sec, Sants |
| **LGBTQ+ community** | Eixample ("Gaixample") |
| **Quiet residential** | Sarrià, Horta, Les Corts |

### Step 3: Check Commute Time

Use TMB App or Citymapper. Barcelona metro is efficient — most central areas are 20-30 min apart.

## Rental Search Tips

### Platforms

| Platform | Best For |
|----------|----------|
| **Idealista** | Largest inventory, agencies + private |
| **Fotocasa** | Good alternative |
| **Badi** | Rooms in shared flats |
| **Spotahome** | Verified, good for foreigners |
| **HousingAnywhere** | Students, mid-term |
| **Facebook groups** | "Pisos Barcelona", "Flats in Barcelona" |

### Red Flags

- Request money before viewing → **SCAM**
- "I'm abroad, wire money" → **SCAM**
- Price too good to be true → **SCAM**
- No real photos → suspicious
- Asking for more than 2 months deposit → against law
- No HUTB number on tourist rental → illegal

### Documents Needed

| Document | Purpose |
|----------|---------|
| **DNI/NIE/Passport** | ID verification |
| **Employment contract** | Proof of income |
| **Last 3 payslips** | Income verification |
| **Bank statements** | Financial stability |
| **Guarantor (aval)** | Often required for foreigners |

### Typical Costs

| Cost | Amount |
|------|--------|
| **Deposit (fiança)** | 1-2 months (max 2 by law) |
| **Agency fee** | 1 month + IVA (if applicable) |
| **First month** | Due on signing |

**Total upfront**: 3-4 months rent

## Barcelona-Specific Issues

### Scarcity
Barcelona has severe housing shortage. Competition is intense. Be ready to:
- Decide quickly (good flats go in days)
- Have documents ready
- Consider short-term first, then search locally

### Illegal Rentals
Many Airbnbs operate illegally. For long-term:
- Check property is registered for residential rental
- Avoid "tourist apartment" for long stays (illegal for tenant)
- Get proper contract (contracte de lloguer)

### Catalan Contracts
Contracts may be in Catalan. This is legal. Request Spanish version if needed, or use translation.
